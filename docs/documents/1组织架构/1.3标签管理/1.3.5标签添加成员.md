# 1.3.5标签添加成员

【1】功能描述：向标签内添加成员

【2】操作流程：
![图片](~@img/1/1.3.5_p1.png)

【3】前置条件：管理员具有标签管理的操作权。

【4】操作步骤：1）管理员登录后台；2）进入组织架构-标签管理页面；3）选中要添加成员的标签；4）点击添加成员；5）选择要添加的成员后点击“确认”

【5】操作界面：
![图片](~@img/1/1.3.5_p2.png)
选中要添加成员的标签，点击添加成员

![图片](~@img/1/1.3.5_p3.png)
选择要添加的成员，点击保存

![图片](~@img/1/1.3.5_p4.png)
添加成功
